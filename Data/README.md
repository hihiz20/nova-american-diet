# Data

The dataset used in this project is from USDA FoodData Central (Branded Foods, December 2025 release).

## Why the data is not included
The dataset exceeds GitHub's file size limits (~1.96M rows) and cannot be committed directly to this repository.

## Download Instructions
1. Go to https://fdc.nal.usda.gov/download-datasets
2. Download the "Branded Foods" dataset
