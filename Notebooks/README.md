# Notebooks

This folder contains Jupyter notebooks focusing on data ingestion, exploratory data analysis (EDA), and NOVA classification. 
Python was used before SQL due to the complexity and disorganized nature of the dataset pulled from USDA FoodData Central. 
The raw data needs processing before querying. Through Exploratory Data Analysis (EDA), I am able to use Python as a "swiss army knife" on the data — checking for nulls, inconsistent data, string consistency, misplaced data, duplicates, and more that SQL cannot easily perform. Additionally, visualizations were not added in this folder. 
Please proceed to the "Visuals" folder.
