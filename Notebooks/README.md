# Notebooks

This folder contains Jupyter notebooks used for data ingestion, exploratory data analysis (EDA), and NOVA classification. Python was used first before SQL is due to the complexity and unorganized dataset that was pulled from the USDA FoodData. The raw data needs processing before querying. Through Exploratory Data Analysis (EDA), I check for nulls, inconsistent data, string consistency, misplaced data, duplicates, and more.
